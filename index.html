Back-End e Front-End CRUD de usuários

Este é um CRUD de usuários com autenticação e controle de acesso. O back-end foi desenvolvido no Node utilizando o framework Express pra criar o servidor, JWT pra fazer a parte de autenticação e o Postgresql como banco de dados SQL. O front-end foi desenvolvido com o framework React.js. Este CRUD de usuários é parte de um projeto maior, ao final teremos um blog completo com cadastro e autenticação de usuários, artigos e categorias de artigos.

O projeto está hospedado no GitHub, lá você encontra a descrição completa, como instalar, executar e usar todas as funcionalidades. Confira.
