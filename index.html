Página Somente HTML

Essa página foi feita só com HTML, o objetivo foi utilizar a maior variedade possível das tags HTML e com a finalidade para a qual foram criadas, de manera bem estruturada, sem gambiarras de estilo ou terrores de semântica. Na verdade o código chega a ser mais interessante que a página em si.

Não estava planejado mas pra deixar então a página mais interessante que o código (e como um ótimo desafio) fiz um estilo CSS pra ela sem mexer em nem uma tag. O estilo foi inspirado no Windows 95. Confira então a versão com CSS e o repositório no GitHub.

Weblogz

Este é um blog multi autor completo com cadastro de usuários, artigos e categorias de artigos. O back-end foi desenvolvido no Node utilizando o framework Express pra criar o servidor, JWT pra fazer a parte de autenticação e o Postgresql como banco de dados SQL. O front-end foi desenvolvido com o framework React.js. Confira a demo do projeto clicando aqui ou na figura.

Este foi um puta primeiro projeto de javascript/React.js, englobou tanto conhecimento front-end quanto back-end. Foram meses de muito conhecimento adquirido, o projeto abrangeu banco de dados, segurança, estilo, autenticação, controle de acesso, componentização, etc. e o mais importante a vivência de um projeto relativamente grande do início ao fim. Da ideia a hospedagem.

Aluracord Retro City

Aqui temos um chat que envia e recebe mensagens em tempo real, as mensagens ficam armazenadas num banco de dados no Supabase e sempre que chega uma nova mensagem no banco de dados todas as telas se atualizam. A aplicação foi desenvolvida com Next.js e SkynexUI. Confira o projeto e o repositório.

O projeto foi desenvolvido durante a Imersão React da Alura em janeiro de 2022, essa é uma modificação do projeto original com novas funcionalidades. Foi um aprendizado prático acerca de React, Next e SkynexUI.

Local Storage Tasks

Este é um gerenciador de tarefas onde é possível organizar suas tarefas diárias, priorizá-las e monitorar seu progresso. Todos os dados são salvos no próprio navegador do usuário (no Local Storage) não necessitando, assim, de um banco de dados. O projeto foi desenvolvido a partir do Vite com Vue 3 e Typescript. Foi utilizado ainda Vue Router para tratar das URLs, Pinia para gerenciamento de estado no Vue, o pré processador de CSS Stylus, e Dice Bear para gerar avatars de usuário.
